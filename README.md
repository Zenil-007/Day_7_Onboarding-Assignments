# Day_7_Onboarding-Assignments
Developer Onboarding Kit – Python environment validation tool with virtualenv checks, linting configuration, CLI support, disk monitoring, and automated setup reporting.

🛠 Developer Onboarding Kit

This project is a Python-based developer environment validation tool designed to automate setup verification for new developers.

It simulates a real-world onboarding scenario where a new team member runs a script to verify their development environment before starting work.

🎯 Features

✅ Python version validation (>= 3.10 required)

✅ Virtual environment detection

✅ Installed package inspection

✅ Tool verification (pylint, black, numpy, requests)

✅ Internet connectivity check

✅ Disk space monitoring (warns if < 1GB free)

✅ Execution time measurement per check

✅ Summary report generation (setup_report.txt)

✅ CLI support:

--verbose → show installed packages
